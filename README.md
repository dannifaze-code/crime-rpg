# Heatline:Underworld

A browser-based crime RPG game built with Vue.js, Three.js, and Firebase.

## Features

- **Turf Wars** - Capture and defend territory on an interactive map
- **Real-time Combat** - Engage in battles with dynamic sprite animations
- **Character Progression** - Level up, earn money, and increase your reputation
- **Global Chat** - Communicate with other players in real-time
- **Leaderboards** - Compete for the top position
- **Weather System** - Dynamic weather effects with Three.js
- **Secure Economy** - Firebase-backed player data and secure transactions

## Tech Stack

- **Frontend**: Vue.js, Three.js, vanilla JavaScript
- **Sprites**: Top-Down Survivor asset pack
- **Backend**: Firebase Realtime Database & Authentication
- **Hosting**: GitHub Pages
